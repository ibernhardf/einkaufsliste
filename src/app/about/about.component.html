Über diese App

Diese Web-App ist im Rahmen des Kurses Web-Applikationen mit Angular von DIB (bildungskarenz-digital.at) im September/Oktober 2024 entstanden und wurde von Bernhard Fischer-Sengseis entwickelt.

Sie dient als Demonstration der erlernten Kenntnisse im Kurs und implementiert unter anderem die folgenden Kursinhalte:

  • Property Binding
  • Event Handling
  • Strukturelle Direktiven
  • Routing
  • Lifehooks (ngOnInit)
  • Services (ProductService, ShopService, SupabaseService)
  • Supabase-Datenbank als Backend

Der vollständige Quellcode der App kann auf Github heruntergeladen werden:
https://github.com/ibernhardf/einkaufsliste
